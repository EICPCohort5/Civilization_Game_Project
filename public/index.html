List of Routes and Their Functions

  • /games/{id} returns a json object of the game with the given id
  • /games lists all the games
  • /games/? lists a game given a filter
  • /games with a post request adds a game to the list
  • /games/{id} with a patch request changes a games information
  • /games/{id} with a delete request deletes a game from the list
  • /publishers/{id} returns a json object of the publisher with the given id
  • /publishers lists all the publishers
  • /platforms/{id} returns a json object of the platform with the given id
  • /platforms lists all the platforms
