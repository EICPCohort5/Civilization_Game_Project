List of Routes and Their Functions

  • /game/{id} returns a json object of the book with the given id
  • /game lists all the games
  • /game/? lists a game given a filter
  • /game with a post request adds a game to the list
  • /game/{id} with a patch request changes a games information
  • /game/{id} with a delete request deletes a game from the list
